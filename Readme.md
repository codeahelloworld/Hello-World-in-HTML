# Hello World in HTML

### Create a Hello-World WEB PAGE in HTML

+ ##### Create a new file called `index.html`.
+ ##### Add the following code to the file.
``` html
<html>
<head>
    <title>Hello World</title>
</head>

<body>
    <p>Hello World</p>
</body>
</html>
```
+ ##### Double click on the file
